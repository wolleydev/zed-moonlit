# Moonlit - a theme for Zed
A variant of Moonlight adjusted to my personal taste. I removed or reduced the presence of some of the most saturated colors and reduced color variation a little to keep it as soothing as possible.

Forked from https://github.com/Rick-VA/moonlight
itself based on https://github.com/atomiks/moonlight-vscode-theme

The theme is adjusted to web frontend work (mostly Vue, Typescript, JSON).
